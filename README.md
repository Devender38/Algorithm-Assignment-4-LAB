## 📌 Overview

This project uses **Backtracking**, **Branch and Bound**, and **String Matching** to solve crew scheduling and pattern search problems efficiently.

---

## 🧠 Algorithms

* Backtracking – explores all valid solutions
* Branch & Bound – optimizes using pruning
* String Matching – Naive (O(n*m)) and KMP (O(n+m))

---

## ⚙️ Tech Used

* Python
* matplotlib, memory_profiler

---

## 🚀 Setup

```bash
pip install -r requirements.txt
jupyter lab
```

---

## 📊 Observations

* Backtracking is slow for large inputs
* Branch & Bound improves performance
* KMP is faster than Naive

---

## 📈 Conclusion

Efficient algorithms are important for solving complex (NP-hard) problems.
